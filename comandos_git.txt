>> Comandos úteis
>> Comando para configurar um projeto local com um remoto no GitHub
git remote add origin https://github.com/valmirfd/valmirfd.github.io.git
->* vai ter essa ligação:[
origin  https://github.com/valmirfd/valmirfd.github.io.git (fetch)
origin  https://github.com/valmirfd/valmirfd.github.io.git (push)
]
--------------------------------------------------------------------------------------|
>> Comando para ver se tem conexão com um repositório remote
git remote -v
--------------------------------------------------------------------------------------|
>> Comando para excluir as credenciais no VS Code
git config --unset credential.helper
--------------------------------------------------------------------------------------|
>> Comando para salvar as credenciais no VS Code
git config credential.helper store
--------------------------------------------------------------------------------------|
>> vários comandos do projeto CodeIgniter novo
git branch ci_01 && git checkout ci_01 && git add . && git commit -m "ci_01"
--------------------------------------------------------------------------------------|
>> Comando para listar branch
|----------------------------------------------------------------|
-> git branch
|----------------------------------------------------------------|
>> Comando para criar uma nova branch
|----------------------------------------------------------------|
-> git branch <nome> 
|----------------------------------------------------------------|
>> Comando para navegar entre as branchs
|----------------------------------------------------------------|
-> git checkout <nome-branch> 
|----------------------------------------------------------------|
>> Comando para unir as branchs
|----------------------------------------------------------------|
-> git merge <nome-branch-que-foi-trabalhada> 
|----------------------------------------------------------------|
>> Comando para excluir uma branch
|----------------------------------------------------------------|
-> git branch -d <nome-branch> 
|----------------------------------------------------------------|
>> Comando para listar commit em uma linha
|----------------------------------------------------------------|
-> git log --oneline
|----------------------------------------------------------------|
>> Comando para add + commit 
|----------------------------------------------------------------|
-> git commit -am "Já add e comita ao mesmo tempo"
|----------------------------------------------------------------|
